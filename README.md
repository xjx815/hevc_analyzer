# HEVC 码流分析工具

这是一个适用于 macOS 的 HEVC (H.265) 码流分析工具，可以详细分析 HEVC 码流的各种信息，包括 SPS、PPS、帧信息和宏块等。

## 功能特点

- **NAL 单元解析**：解析码流中的每个 NAL 单元，识别其类型和属性
- **SPS 分析**：提取序列参数集信息，包括分辨率、帧率、编码配置等
- **PPS 分析**：提取图像参数集信息，包括参考帧配置、量化参数等
- **帧信息分析**：分析每个帧的类型、序号、参考关系等
- **宏块分析**：分析宏块的编码模式、预测方式、运动矢量等
- **码流摘要**：生成码流的统计信息，包括 NAL 单元数量、帧数量等

## 构建方法

### 前提条件

- macOS 操作系统
- CMake 3.10 或更高版本
- C++ 编译器 (支持 C++11 标准)

### 构建步骤

1. 克隆项目到本地
2. 创建构建目录并进入
3. 运行 CMake 生成构建文件
4. 编译项目

```bash
# 克隆项目（如果尚未克隆）
git clone <repository_url> hevc_extractor
cd hevc_extractor

# 创建构建目录并构建
mkdir -p build
cd build
cmake ..
make

# 构建完成后，可执行文件将位于 bin 目录
```

## 使用方法

### 基本用法

```bash
# 分析 HEVC 码流文件
./bin/hevc_analyzer <hevc_file>

# 示例
./bin/hevc_analyzer test.hevc
```

### 输出说明

工具会输出以下信息：

1. **NAL 单元信息**：每个 NAL 单元的类型、长度等基本信息
2. **SPS 信息**：序列参数集的详细配置，如分辨率、编码模式等
3. **PPS 信息**：图像参数集的详细配置，如参考帧设置、量化参数等
4. **帧信息**：每个帧的类型、序号、参考关系等
5. **宏块信息**：宏块的编码模式、预测方式、运动矢量等（仅显示前几个宏块）
6. **码流摘要**：整个码流的统计信息，如 NAL 单元数量、帧数量等

## 技术实现

### 核心模块

1. **NAL 单元解析**：识别 NAL 单元的起始码，解析 NAL 单元头，提取负载数据
2. **指数哥伦布编码解析**：实现指数哥伦布编码的解码，用于解析 HEVC 中的各种语法元素
3. **SPS/PPS 解析**：根据 HEVC 标准解析序列参数集和图像参数集的各个字段
4. **帧信息分析**：解析帧头部信息，提取帧的类型、序号等属性
5. **宏块分析**：解析宏块的编码信息，包括预测模式、运动矢量等

### 关键算法

- **起始码检测**：通过滑动窗口检测 NAL 单元的起始码 (0x000001 或 0x00000001)
- **指数哥伦布编码解码**：根据前导零的个数确定编码长度，然后解码出实际值
- **位级操作**：精确的位级读取和操作，用于解析 HEVC 码流中的各种语法元素

## 注意事项

1. **文件格式**：工具支持原始 HEVC 码流文件 (.hevc)，不支持封装格式如 MP4、MKV 等
2. **性能考虑**：对于大型码流文件，分析过程可能会消耗较多时间和内存
3. **宏块分析**：当前实现中宏块信息为模拟数据，实际应用中需要根据具体码流进行解析
